## 用自己的话回答

- 什么是特征，什么是标签：特征是模型用来判断的输入信息（如使用月数、投诉次数），标签是要预测的答案（如是否流失）
- 为什么要保留测试集：测试集模拟未见过的数据，用于检验模型泛化能力，避免模型只记住答案而失去实际用途
- 为什么83%准确率仍可能没有用：如果多数类占83%，永远猜多数类也有83%准确率。模型需要比最低参照线做得更好才有实际价值
